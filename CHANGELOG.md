# Changelog

本仓库的重要规则变更记录在这里。

## 2026-04-23

### Changed

- 将 `SKILL.md` 升级为单文件自包含执行规范，Codex 使用时不再依赖 `references/` 中的外部规则文件。
- 将“长文去模板模式”提升为整段、整篇中文优化任务的默认第一步，并明确“先去模板，再评估整体中文生态，最后处理局部病句”的工作顺序。
- 将二元对立话术提升为最高优先级风险项，补充显性结构、隐含结构、替换原则，以及扩充时的反弹警告。

### Added

- 新增完整的 7 步扫描流程：二元对立话术、第二人称教程感、机器式能力清单、抽象过渡句与泛指词、空泛评价词与抽象名词堆叠、过密转场、高风险“把”字句。
- 新增“适用场景”与“核心原则”段落，明确本 skill 同时覆盖去 AI 味、去模板感、去教程感的长文优化任务。
- 新增语言生态原则评估、被动语态与介词修正、弱动词识别与修订、冗余“之一”结构、连词与修饰语规范等后续模块的内联规则说明。
- 新增更新后的 `README.md`，说明 `SKILL.md` 为主规范，`references/` 作为补充材料保留。

### Clarified

- `references/` 目录仍可继续保留，但在当前发布形态中属于补充材料，不再是运行时前置依赖。
- 对内部安全通报、风险提示、风险通告、复训速查卡等任务继续保持保守边界，只删明显模板壳，不改变事实关系。
