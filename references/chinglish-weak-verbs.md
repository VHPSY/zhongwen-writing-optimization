# 中式英语弱动词识别与修订

## 何时使用

当遇到含有泛化"万能"动词（如"作出""进行""加以""予以"）搭配抽象名词（如"作出贡献""进行研究"）的表达时。这种模式通常源自英语句法影响（"make a contribution"、"conduct research"），削弱中文的清晰度与力度。

**不使用场景**：
- 正式科学、技术或法律文件，此类结构对精确性有必要
- 不存在地道单动词替代方案的情况

## 识别目标

| 泛化动词 | 典型搭配 | 问题 |
|----------|----------|------|
| 作出 | 作出贡献、作出决定、作出说明 | 动作被名词化 |
| 进行 | 进行研究、进行试验、进行讨论 | 冗余辅助动词 |
| 加以 | 加以改进、加以分析、加以说明 | 可直接用动词 |
| 予以 | 予以支持、予以处罚、予以考虑 | 法律用语泛化 |

## 修订原则

若中文本身已有现成动词，不要将一个动作拆成"名词 + 模糊动词"。拆分把动态动作变成静态短语——这是恶性西化的核心症状。

**核心规则**：动态动词优先于名词化表达。

## 典型修订对照

| 弱表达 | 推荐表达 | 说明 |
|--------|----------|------|
| 作出贡献 | 贡献很大 / 贡献突出 | 避免将"contribute"名词化 |
| 进行研究 | 研究 / 详加研究 | "研究"本身就是动词 |
| 进行试验 | 试验 / 做实验 | 优先具体动词 |
| 加以改进 | 改进 | 辅助动词冗余 |
| 予以支持 | 支持 | 直接用动词即可 |
| 作出决定 | 决定 | 单动词更简洁 |
| 进行讨论 | 讨论 | 无需辅助 |

## 英语来源对照（供 awareness）

| 英语表达 | 中文弱表达 | 推荐表达 |
|----------|-----------|----------|
| apply pressure | 施加压力 | 施压 |
| give authorization | 给予授权 | 准许 |
| take appropriate action | 采取适当行动 | 采取行动 |
| make an investigation | 进行调查 | 调查 |

## 例外情况

- 法律或监管文本："予以处罚"可保留以维持程序性清晰度
- 高度正式学术语境，名词化服务于修辞平衡——即便如此，仍优先使用动词
