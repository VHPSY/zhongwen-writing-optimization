# 中文连接词与修饰语使用规范

## 何时使用

- 文本中存在多个同类词或短语并列（名词、形容词等）
- 出现连续多个"的"字修饰结构
- 修饰语较长导致句子头重脚轻或理解困难
- 表达中出现学术化、冗余或不自然的修饰形式

> 注意：本技能不适用于需要明确标示逻辑转折、因果等关系的场景。

## 执行步骤

### 1. 处理并列成分

- **同类词直接并列，省略连接词**：
  - ✅ "夫妻"、"你我"、"前后"、"金木水火土"
  - ✅ "柴米油盐酱醋茶"
  - ❌ "柴、米、油、盐、酱、醋以及茶"

- **若必须使用连接词**，优先选用"而""并""而且"，避免滥用"及""以及"：
  - ✅ "慢而可靠"
  - ❌ "慢及可靠"

- **并列项数量指南**：
  - ≤3 项：可直接并列，无需连接词
  - >3 项：若需停顿可用顿号，但避免末尾加"及"或"以及"

### 2. 优化"的"字使用

- 避免连续使用多个"的"字
- 可用"而"连接形容词，或重构为更简洁表达

| 原句 | 优化 |
|------|------|
| 参差的斑驳的黑影 | 参差而斑驳的黑影 / 斑驳参差的黑影 |
| 有不洁的颜色的都市的河沟 | 都市脏河沟 |
| 白色的鸭 | 白鸭 |
| 热情型的人 | 热情的人 |
| 太专业性的字眼 | 太专门的字眼 |
| 具有高度复杂性的系统 | 极其复杂的系统 |
| 难度很高的 | 很难的 |

### 3. 选择前饰或后饰

- **短修饰语（≤4 字）**：前饰，自然流畅
  - ✅ "热情的人"
  - ✅ "白色的纸"

- **长修饰语（>4 字或含从句）**：后饰，增强句子弹性
  - 不佳："站在门口、穿着红色外套、手里拿着一本书的男人"
  - 推荐："我见到一个男人，站在门口，穿着红色外套，手里还拿着一本书"

### 4. 消除伪学术化修饰

| 冗余表达 | 推荐表达 |
|----------|----------|
| 难度很高的 | 很难的 |
| 热情型的人 | 热情的人 |
| 太专业性的字眼 | 太专门的字眼 |
| 具有高度复杂性的系统 | 极其复杂的系统 |
| 做一个决定的 | 决定的 |

执行结果应使语言更符合中文表达节奏，提升可读性与自然度。
