# 中文被动语态与介词滥用修正

## 何时使用

- 句子包含"被"字且非强调受事者或施事者未知的特殊情况
- 出现冗余介词结构，如"有关……""关于……""由于……使得……"
- 文本为叙述句、说明文或新闻报道，需符合中文自然表达习惯
- 用户希望提升语言简洁性与地道性，避免西式语法痕迹

## 执行步骤

### 1. 被动语态修正

**优先转为主动表达**：中文倾向主动句式，避免"被"字句。

| 原句 | 修正 |
|------|------|
| 他被怀疑偷东西 | 他有偷东西的嫌疑 |
| 他被选为议长 | 他当选为议长 |
| 窗户被打碎了（保留：施事者未知） | — |
| 受害者被救出（保留：强调受事者） | — |

**使用自然被动动词替代**：如"遇害""当选""获准""蒙受"等。

**省略施事者，用主动形式表被动含义**：
- "糖吃光了"
- "稿写了一半"
- "书读完了"

### 2. 介词滥用修正

**删除冗余"有关""关于"**：
- "有关文革的种种" → "文革种种"
- "关于这件事情的处理" → "这件事的处理"

**拆解"由于……使得……"结构**：改用因果隐含或直接陈述。
- "由于家境贫穷，使得他休学" → "他家境贫穷，只好休学"
- "由于天气恶劣，使得比赛延期" → "天气恶劣，比赛延期"

**简化不自然介词短语**：
- "来到我们的中间" → "来到我们中间"
- "对于这件事的看法" → "对这件事的看法" 或 "这件事的看法"

### 3. 核心原则

- 中文重**意合**（靠语境传递逻辑），不重**形合**（靠连接词显性标记）
- 仅保留对语义有实质贡献的介词；其余应删除以提升流畅度
- 若句子需强调受事者或施事者确实未知，可保留被动结构
