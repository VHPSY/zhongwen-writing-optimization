# zhongwen-writing-optimization

中文写作优化技能集。用于识别并修正中文表达里的西化痕迹、模板化表达与教程感，包括冗余结构、弱动词、被动句、介词堆砌、修饰语失衡，以及公众号长文、观点稿、整理稿中常见的二元对立话术、第二人称引导、能力清单和高频“把”字句。

当前版本采用“主规范内联”的发布方式：`SKILL.md` 已包含完整执行规则，Codex 使用时不再依赖外部引用文件才能完成核心判断。`references/` 目录保留为补充材料与历史拆分版本，便于查阅和继续演化。

## 仓库内容

- `SKILL.md`：完整执行规范，包含默认工作顺序、长文去模板 7 步扫描、语言生态评估、局部病句修订、Ban List 和保守边界
- `references/`：补充参考材料，保留原先拆分出的专题说明
- `CHANGELOG.md`：规则变更记录

## 适用场景

- 学术写作、商务沟通、新闻报道、说明文
- 日常中文表达的地道化、简洁化和逻辑优化
- 用户提交整段、整篇文本，希望系统性优化中文表达
- 用户明确要求“去 AI 味”“去模板感”“去教程感”
- 文本同时存在多种问题，不适合只靠单一规则处理

## 这个 skill 的特点

- 以“长文去模板优先 + 局部病句细修”为核心顺序
- 将二元对立话术视为最高优先级模板风险
- 兼顾整体中文生态与具体语言病症
- 适合整段评估，也适合按症状精修
- 对“内部安全通报”这类任务内置保守口径约束

## 使用方式

1. 将 `SKILL.md` 放入你的 skill 目录。
2. 遇到整体中文优化任务时直接触发主 skill。
3. 如需查看分项依据，再查阅 `references/` 下的补充材料。

## 适合发布的原因

- `SKILL.md` 已自包含核心规则，便于直接分发和复制使用
- 内容自洽，不依赖本地脚本或私有资源
- 可单独使用，也可作为中文写作优化 skill 集合的一部分
